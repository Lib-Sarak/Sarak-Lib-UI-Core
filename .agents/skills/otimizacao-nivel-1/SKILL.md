# Otimização de Performance: Nível 1

Esta skill é responsável por aplicar otimizações técnicas de performance de alto impacto, focando exclusivamente em soluções de custo zero (recursos nativos do navegador e ferramentas gratuitas) sem degradação da qualidade visual perceptível.

## Principais Áreas de Atuação:
- Conversão de Ativos (Imagens/Vídeos)
- Estratégias de Cache e Armazenamento Local
- Eficiência na Entrega de Código (Bundle/Hydration)
- Otimização de Core Web Vitals (LCP, CLS, TBT)
