# Skill: Criar Skills — Referência Estrutural

## O que é
Esta é a skill de **meta-criação**. Ela define como qualquer nova skill deve ser construída — estrutura, profundidade, padrão de escrita, exemplos e templates obrigatórios.

Toda skill criada sem seguir esta referência está incompleta e não deve ser usada.

## Objetivo
- Ser a fonte de verdade para a criação de novas skills.
- Garantir que qualquer skill produzida seja suficientemente rica para que qualquer agente (LLM) consiga executá-la com precisão sem intervenção humana constante.
- Padronizar a estrutura e o nível de profundidade de todas as skills do sistema.

## Responsabilidades Exclusivas desta Skill
1. Definir e documentar a estrutura obrigatória de toda skill.
2. Fornecer templates prontos para uso de cada arquivo.
3. Definir critérios de qualidade para considerar uma skill "completa".
4. Servir de checklist de validação antes de qualquer skill ser registrada como operacional.

## Quando usar
- Ao criar qualquer nova skill do zero.
- Ao revisar uma skill existente que parece incompleta ou ambígua.
- Quando um agente precisar decidir se uma skill está pronta para uso.
