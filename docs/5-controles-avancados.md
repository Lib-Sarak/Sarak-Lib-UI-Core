# ⚙️ Controles Avançados (Scaling e Densidade)

A Sarak UI foi desenhada para se adaptar a qualquer monitor, desde notebooks de 13" até telas ultra-wide de 34", sem perda de legibilidade.

## 1. Viewport Scaling (O Motor de Escala)
Através da variável principal `--font-size-factor`, a interface inteira pode ser redimensionada proporcionalmente.
- **N Níveis de Escala:** Opções como **P1 (Extra Small)** até **G1 (Extra Large)**.
- Ao mudar a escala, não apenas o texto aumenta/diminui, mas também as margens, ícone e o espaçamento entre colunas, mantendo a proporção visual do design.

## 2. Densidades Dinâmicas
A densidade controla o "respiro" da aplicação e a economia de espaço em tela:
- **Compact:** Ideal para analistas e operadores de dados que precisam de máxima informação em pouco espaço.
- **Standard:** O padrão Sarak para uso diário.
- **Comfortable:** Um modo voltado para consumo de conteúdo e leitura focada.

## 3. Atalhos Globais (Shortcuts)
O sistema Sarak possui uma engine de escuta de teclado que permite realizar ações rápidas sem usar o mouse:
- **Alternar Tema:** Mude a personalidade visual instantaneamente.
- **Modo Foco:** Esconda a navegação para ampliar a área de trabalho.
- **Busca e Ação:** Atalhos para abrir janelas de busca ou o Painel de Customização.
